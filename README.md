# AI Sign Language Translator

This project builds a deep learning model to recognize sign language gestures and convert them into speech.

## Features

- Hand gesture recognition using CNN
- Image preprocessing with OpenCV
- Speech output using gTTS
- ASL alphabet classification

## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- gTTS

## Workflow

Image → CNN Model → Letter Prediction → Text → Speech

## Dataset

ASL Alphabet Dataset from Kaggle

## Future Improvements

- Real-time webcam detection
- Sentence generation
- MediaPipe hand detection
